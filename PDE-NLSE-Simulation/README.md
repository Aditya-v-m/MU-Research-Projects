# Numerical Methods for the Nonlinear Schrödinger Equation (NLSE)

## Overview
This project studies numerical methods for solving NLSE and GPE equations.

## Methodology
- Finite Difference Methods
- Spectral Methods

## Key Concepts
- Partial Differential Equations
- Numerical simulation

## Applications
- Quantum physics
- Nonlinear systems
