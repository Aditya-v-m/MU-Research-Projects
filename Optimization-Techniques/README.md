# Optimization Techniques for Transportation Cost Minimization

## Overview
This project develops a Linear Programming (LP) model to minimize transportation costs under operational constraints. The study formulates and solves a constrained optimization problem relevant to real-world logistics systems.

## Problem Formulation
A transportation company must allocate vehicles across multiple routes while minimizing total cost, subject to:
- Capacity constraints
- Resource limitations
- Demand requirements

## Methodology
- Mathematical formulation of Linear Programming Problem (LPP)
- Graphical solution method (for low-dimensional cases)
- Two-Phase Simplex Method for general feasibility and optimality

## Key Concepts
- Objective Function Optimization
- Feasible Region Analysis
- Constraint Satisfaction
- Linear Programming Theory

## Applications
- Transportation and logistics optimization
- Supply chain cost minimization
- Resource allocation in operations research
